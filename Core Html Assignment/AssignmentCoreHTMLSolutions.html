Question No.8

NodeJS
Node.js is a back-end JavaScript runtime environment, runs on the V8 JavaScript engine, and executes JavaScript code outside a web browser. Node.js lets developers use JavaScript to write command line tools and for server-side scripting.
ReactJS
React.js is a free and open-source front-end JavaScript library for building user interfaces based on components. It is maintained by Meta (formerly Facebook) and a community of individual developers and companies. React can be used to develop single-page, mobile, or server-rendered applications with frameworks like Next.js.
Angular
Angular is an open-source framework developed by Google, it helps developers build single-page web applications. One of the benefits of using Angular is that it uses TypeScript, a superset of JavaScript that can help improve code quality. Moreover, Angular includes several features that make development faster and easier, such as an automated form validation system.
Django
Django is one of the best frameworks, it helps developers quickly create web applications without worrying about low-level details such as creating HTML templates. Django includes several features that are essential for modern Web Development training, including a template engine for creating HTML views, and a wide variety of tools and libraries for common tasks.
Bootstrap
Bootstrap is a free, open-source framework that includes HTML, CSS, and JavaScript. It is easy to use and helps developers quickly create responsive websites. One of the benefits of Bootstrap is that it includes several built-in components, such as buttons and navigation bars. This makes it quick and easy to create consistent, reliable web designs.

Question No.9

  1. BOLD Tag
    • Specific use:- This tag is use to draw the reader's attention to the element's contents, which are not otherwise granted special importance.
    • Best Practice:- Use strong {strong} tag. When bolding text, it is considered a best practice to use the strong tag. This is because it is a semantic element.
  2. ITALICS Tag
    • Specific use:- The italics tag defines a part of text in an alternate voice. The content inside is typically displayed in italic.It is often used to indicate a technical term
    • Best Practice:- Use emphasis {em} tag. The “em” in the tag literally stands for emphasis. Browsers will, by default, make italicize text that is wrapped in HTML emphasis tags.
  3. STRIKE-THROUGH Tag
    • Specific use:- This HTML element renders text with a strikethrough, or a line through it
    • Best Practice:- Use Delete {del} tag. The delete tag represents a range of text that has been deleted from a document. This can be used when rendering "track changes" or source code diff information.
  4. SUPERSCRIPT Tag
    • Specific use:- This HTML element defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.
    • Best Practice:- Just both the opening and closing tag is mandatory while using the Superscript tag.
  5. MARK Tag
    • Specific use:- The mark tag is used to highlight some part of the text for quick reference inside another element
    • Best Practice:- It defines text that should be marked or highlighted.
